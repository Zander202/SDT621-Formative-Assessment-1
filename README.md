📘 C# Multi-Project Repository

This repository contains multiple C# projects organized by sections. Each folder represents a separate application or assignment.

📂 Project Structure
SectionA_Q1_StudentResults → Student results 
SectionA_Q2_ATMSystem → ATM 
SectionA_Q3_ProgrammingLang... → Programming language add and delete
SectionB_Q1_EmfuleniMunicipality → Municipality system
SectionB_Q2_HomeAffairsProcess... → Home Affairs Form

Each folder is an independent C# project and can be run separately.

⚙️ Requirements

Make sure you have the following installed:

Visual Studio (recommended)
.NET SDK (version depends on the project, usually .NET 6 or later)
🚀 How to Run the Projects
Download it Unzip it and Run your desired project
